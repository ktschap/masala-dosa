  1. How do I upload class flyer files?

    Note that uploaded files must be less than 1 MB in size. From the top menu of the admin pages, select Class Flyers. This page allows you to view flyers currently available in the system, and delete any of those that are no longer needed. To upload a new flyer, enter a name for the flyer, and click the browse button to locate the file on your local machine. It is strongly recommended to choose a flyer name that will allow you to identify its semester/year, and class. This will allow you to clearly relate the named flyer to the class, and to determine whether it is currently needed or is from a previous semester and can be permanently deleted.

    Once the flyer name is filled in, click "Import Flyer" to upload.

  2. How do I add the flyers to a class?

    When editing a class description on the classes page, the name of the flyer you uploaded should appear as a choice in the Choose Uploaded Flyer field. Simply change the selection and click the Save button.

    Back to Contents.

  3. How do I open/close class registration?

    From the top menu of the admin pages, select Site Configuration. A checkbox labeled Registration On? controls registration. If checked, users can register. Make any changes and click Save.

    Back to Contents.

  4. How do I change the semester and begin building a new class list for a new semester?

    From the top menu of the admin pages, select Site Configuration. A dropdown list controls the current semester, make your change and click Save. Note that changing the semester hides all class information and registration information, on both the public registration pages.
